-- Installation d'une Machine Virtuelle --

Étape 0 : S’assurer que la virtualisation est bien activée sur la machine physique

Il est important de s’assurer que la virtualisation est activée sur la machine physique. Si ce n'est pas le cas, il faudra l'activer dans le BIOS.

Étape 1 : Télécharger un logiciel de virtualisation

Dans un premier temps, il faut télécharger un logiciel de virtualisation. Plusieurs options existent, telles que VMware Workstation Player ou QEMU, mais nous privilégions VirtualBox Oracle pour son interface intuitive.

Rendez-vous sur https://www.virtualbox.org/wiki/Downloads et choisissez la version correspondant à votre système d’exploitation actuel (Windows dans notre cas).

Étape 2 : Télécharger un fichier .iso

Ensuite, il faut télécharger le fichier .iso nécessaire pour fournir à la machine virtuelle les fichiers d’installation du système d’exploitation. Nous choisissons ici Ubuntu, mais vous pouvez aussi utiliser Debian ou d'autres distributions Linux.

Rendez-vous sur https://ubuntu.com/download/desktop pour télécharger le fichier .iso d'Ubuntu.

Étape 3 : Configuration de la machine virtuelle

Étape 3.1 : Lancer Oracle VirtualBox

Une fois le téléchargement terminé, lancez Oracle VirtualBox. Vous arriverez sur l'interface suivante (qui peut légèrement varier selon la version de l'application).

Ensuite, cliquez sur le bouton bleu "Nouvelle" en haut de l’interface pour créer une nouvelle machine virtuelle.

Étape 3.2 : Renseigner les informations de la machine virtuelle

Une fenêtre s’ouvrira et vous devrez entrer les informations de base pour votre machine virtuelle : son nom, l’emplacement où elle sera stockée sur votre machine, et le fichier ISO téléchargé précédemment.

Si vous avez choisi Ubuntu, le type et la version se configureront automatiquement sur Linux et Ubuntu (64-bit).

Étape 3.3 : Configurer le hardware de la machine virtuelle

Il ne vous reste plus qu’à définir un nom d'utilisateur, un mot de passe, ainsi qu’un nom d’hôte pour la machine virtuelle. Ensuite, vous devrez allouer de la mémoire vive et des processeurs à la machine.

ATTENTION : Ne donnez pas trop de ressources à la machine virtuelle, cela pourrait ralentir votre machine physique et virtuelle.

Vous pourrez aussi choisir si vous voulez créer un disque dur virtuel, qui sera plus portable mais plus lent, ou bien utiliser le disque dur de la machine physique, qui est plus rapide mais moins portable.
