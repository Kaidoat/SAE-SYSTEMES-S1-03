-- Résumé sur la Virtualisation --

Définition de la virtualisation

La virtualisation est une technologie qui permet de créer des versions virtuelles (ou simulées) de ressources physiques...

Il existe différentes formes de virtualisation, telles que :

  • La virtualisation de serveur
  • La virtualisation de stockage
  • La virtualisation de réseau

Différence entre émulateur, simulateur et outils de virtualisation

  • Émulateur : Reproduction exacte d’un système dans un environnement différent.
  • Simulateur : Reproduction approximative d’un comportement ou d’un processus sans reproduire entièrement le système.
  • Virtualisation : Création d’environnements isolés sur un même matériel physique, permettant l'exécution de multiples systèmes ou applications en parallèle.

Type de virtualisation pour les environnements

  • VirtualBox : Virtualisation de type 2 (hébergée).
  • VMware Workstation : Virtualisation de type 2 (hébergée).
  • Stratus : Virtualisation de type 1 (bare-metal).

Forme/Lieu de sauvegarde de la machine virtuelle créée

VirtualBox/VMware Workstation

  • Fichiers de disque virtuel (VHD, VMDK) et paramètres de configuration.
  • Stockage sur disque local ou support externe.

Stratus

  • Sauvegardes via snapshots ou réplication, stockées sur RAID ou serveur distant.

Choix techniques possibles et justification

  • CPU : Nombre de cœurs en fonction des besoins (1-2 pour tests, 4+ pour performances).
  • RAM : 2 Go (Linux léger) à 8 Go (Windows ou logiciels gourmands).
  • Disque : Taille dynamique (optimise espace) ou fixe (réserve).
  • Réseau :
    • NAT (accès Internet sécurisé).
    • Bridge (connexion directe au réseau).
    • Host-only (isolé pour tests internes).
  • Périphériques : Activer USB ou partage de dossiers si nécessaire.
